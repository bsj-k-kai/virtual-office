# バーチャルオフィス

リモートでもオフィスにいるように、2D空間を歩き回りながら同僚に気軽に話しかけられる Web アプリです。

## 主な機能

- **2Dオフィスマップ** — デスク・会議室・ラウンジ付きの共有空間
- **リアルタイム同期** — 複数人が同時に入室し、位置が即時反映
- **近接音声** — 一定距離内に近づくと WebRTC で音声が自動接続（Gather 風）
- **ステータス** — 「対応可」「取り込み中」「離席中」（取り込み中は話しかけ不可）
- **Google カレンダー連携** — 勤務用以外のカレンダーで、いま参加中の予定を吹き出しに表示

## 技術スタック

| 層 | 技術 |
|---|---|
| フロント | React + TypeScript + Vite |
| リアルタイム | Socket.io |
| 音声 | WebRTC (P2P) + STUN |

## ログイン

## 本番デプロイ

他のユーザーと利用するには HTTPS 上にデプロイしてください。手順は **[DEPLOY.md](./DEPLOY.md)** を参照（Render / Railway / Docker 対応）。

```bash
npm run build
NODE_ENV=production ENABLE_DEMO_BOTS=false npm start
# → https://your-domain/ をチームに共有
```

## セットアップ（開発）

```bash
cd /Users/k.kai/project/virtual-office

# 依存関係インストール
npm install
cd client && npm install && cd ..
cd server && npm install && cd ..

# 開発サーバー起動（クライアント + サーバー同時）
npm run dev
```

- クライアント: http://localhost:5173
- サーバー: http://localhost:3001

## デモメンバー（ボット）

サーバー起動時に、動作確認用の5人が自動でオフィスにいます。

| 名前 | 初期位置 | ステータス |
|------|----------|------------|
| 田中 太郎 | デスク A 付近 | 対応可（歩き回る） |
| 鈴木 花子 | デスク B 付近 | 対応可（歩き回る） |
| 佐藤 健太 | デスク C 付近 | 取り込み中（ゆっくり移動） |
| 山田 美咲 | デスク F 付近 | 対応可（歩き回る） |
| 伊藤 大輔 | ラウンジ付近 | 離席中（動かない） |

🤖 デモメンバーに近づくと、テスト用のループ音楽が流れます（離れるとフェードアウトして停止）。距離に応じて音量が変わります。

## 使い方

1. ブラウザで http://localhost:5173 を開く
2. 表示名を入力して「オフィスに入る」
3. マップ上にデモメンバーが表示されます。近づいて近接UIを確認できます
4. **音声の確認**は別タブや別ブラウザでもう1人入室してください
4. 矢印キー / WASD で移動し、相手の近くに寄ると音声がつながります
5. 初回はマイクの許可を求められます
